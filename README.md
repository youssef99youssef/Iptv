## 🚀 كيفية البدء

### طريقة 1: فتح مباشر في المتصفح
1. انسخ الملفات إلى مسار محلي
2. افتح `index.html` مباشرة من المتصفح

### طريقة 2: استخدام خادم محلي

```bash
# إذا كان لديك Python 3
python -m http.server 8000

# أو إذا كان لديك Python 2
python -m SimpleHTTPServer 8000

# أو باستخدام Node.js
npx http-server

# ثم افتح المتصفح على:
# http://localhost:8000
```

## 🔧 التخصيص

### تغيير الألوان
عدّل القيم في `:root` في ملف `style.css`:

```css
:root {
    --primary-color: #FF6B35;      /* اللون الأساسي */
    --secondary-color: #004E89;    /* اللون الثانوي */
    --accent-color: #00A8E8;       /* لون مميز */
    /* ... ألوان أخرى */
}
```

### تعديل المحتوى
- عدّل النصوص مباشرة في `index.html`
- غيّر الأسعار والخطط
- أضف أو احذف الأسئلة الشائعة

### إضافة تكامل API
يمكنك تعديل `script.js` لإضافة:
- معالجة نموذج التواصل
- دمج بوابات الدفع
- تسجيل المستخدمين
